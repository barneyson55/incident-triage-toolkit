# AI_EDIT_POLICY.md

## NO-AI zones (never edit)
- `.env` and any `.env.*` (except `.env.example`)
- `.venv/`
- `dist/`, `build/`
- `.ruff_cache/`, `.pytest_cache/`
- Generated outputs (do not commit): `parsed.json`, `timeline.md`, `runbook.md`

## Allowed
- All source code under `triage_toolkit/**`
- Tests under `tests/**`
- GitHub workflows under `.github/**`
- Documentation under `docs/**` and `README.md`
- Tooling files like `Makefile`, `requirements*.txt`, `pyproject.toml`

## No signature / watermark rule
- Do NOT add: "generated by", "created by AI", model names, or any watermark.

## If a task requires touching NO-AI zones
- STOP and add a checkbox item to `docs/user_todo.md` explaining what is needed.
